# Number Analyzer (Day 10 Project)

A small Python project that analyzes a sequence of user-entered numbers using **functions**, **list comprehensions**, and **generator expressions**.  
Built as part of my daily Python learning journey.

---

## Features

This program:

- Accepts a list of numbers from the user  
- Returns a breakdown of:
  - All even numbers  
  - All odd numbers  
  - How many numbers are above a given threshold  
  - The sum of all numbers divisible by a given divisor  
- Uses:
  - Pure functions  
  - List comprehensions  
  - Generator expressions  
  - Dictionary-based reporting  
  - Clean modular code structure

##  How to Run

In your terminal or IDE:

```bash
python3 number_analysis.py
```

You will be prompted to:

1. Enter a list of numbers  
2. Enter a threshold  
3. Enter a divisor  

The program prints a formatted analysis report.
